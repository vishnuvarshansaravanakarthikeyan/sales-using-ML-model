# 📊 Sales Data Analysis with Machine Learning

This project is a beginner-friendly Data Science project that analyzes sales data and applies basic Machine Learning models using Python.

---

## 🚀 Features

- 📁 CSV Data Handling using Pandas
- 📊 Data Visualization using Matplotlib & Seaborn
- 🔗 Correlation Analysis (Heatmap)
- 📈 Linear Regression (Revenue Prediction)
- 🤖 Logistic Regression (High/Low Sales Classification)

---

## 📂 Dataset

The dataset contains:

- Date
- Product ID
- Quantity Sold
- Unit Price
- Customer ID

---

## 🧠 Concepts Used

- Data Cleaning & Processing
- Feature Engineering (Revenue Calculation)
- Exploratory Data Analysis (EDA)
- Supervised Machine Learning
